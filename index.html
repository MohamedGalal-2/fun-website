My Portfolio

Project 1: Smart Home Automation System

Developed an embedded system for controlling home devices such as lights, fans, and security systems. This system used sensors for smart control and a microcontroller for efficient power management.

Skills Used: Embedded C, Microcontrollers, IoT, Circuit Design, Sensor Integration

Project 2: Automated Plant Watering System

Created a fully automated plant watering system using soil moisture sensors and a microcontroller. The system waters the plants automatically based on moisture levels, reducing the need for manual intervention.

Skills Used: Embedded C, Microcontrollers, Sensors, Automation, IoT

Project 3: Video Game Console Emulator

Developed a game console emulator that allows classic games to run on modern hardware. The emulator was optimized to work with embedded systems, providing smooth gameplay on limited hardware resources.

Skills Used: Embedded Systems, C Programming, Game Development, Optimization
